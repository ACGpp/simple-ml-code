# 我是第1章的标题
我是第1章的正文，下面给出插入图片的两种方式。

html代码：
```html
<div align=center>
< img width="300" src="./images/1_1.jpeg"/>
</div>
<div align=center>图1.1 我是图片名称</div>
```
效果如下：
<div align=center>
< img width="300" src="./images/1_1.jpeg"/>
</div>
<div align=center>图1.1 我是图片名称</div>

markdown代码：
```markdown
![图1.1 我是图片名称](./images/1_1.jpeg)
```
效果如下：
![图1.1 我是图片名称](./images/1_1.jpeg)
